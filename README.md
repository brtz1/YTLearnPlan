# YTLearnPlan

Local storage repository for a YouTrack Workflow learn plan.

## Structure

- `data/`: stored learn-plan content
- `data/raw/`: source material and unprocessed exports
- `data/notes/`: research notes and working drafts
- `data/lessons/`: lesson plans and curated learning content
- `data/workflows/`: workflow examples, snippets, and references
- `data/templates/`: reusable document templates
- `archive/`: retired or superseded material
- `docs/`: repository documentation and conventions

## 1-Week Sprint Deliverables

This repository now includes a 7-day accelerated study pack for the
YouTrack Internal Automation Engineer path:

- Day 1 YouTrack basics and process modeling notes
- Day 2 JavaScript/TypeScript refresh focused on workflow work
- 4 workflow examples with documentation
- REST API samples and a Node.js integration script
- reusable requirement and design templates
- portfolio and interview-prep documents

## Suggested Use

Work through the materials in this order:

1. `data/notes/day-1-youtrack-basics.md`
2. `data/lessons/day-2-js-ts-basics.md`
3. the workflow examples under `data/workflows/`
4. the integration materials under `data/raw/api-samples/`
5. `docs/portfolio-overview.md`
6. `docs/interview-stories.md`
