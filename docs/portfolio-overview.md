# Portfolio Overview

## What this repo demonstrates

This repository is a 1-week accelerated learning portfolio for a
YouTrack Internal Automation Engineer path. It focuses on the five core areas
that matter most for the role:

- YouTrack product modeling
- JavaScript and TypeScript fundamentals for automation
- workflow design and maintenance
- REST API and integration patterns
- Git-friendly documentation and stakeholder communication

## Included examples

- product and process modeling notes
- 4 small workflow examples
- REST API request and response samples
- a minimal Node.js integration script
- requirement, design, and postmortem templates
- an architecture write-up for a realistic internal automation scenario

## What I can now speak to

- how to model a team process in YouTrack
- when to use fields and configuration versus workflow code
- how to build simple guard and routing workflows
- how to create or update issues through the REST API
- how to explain trade-offs and maintenance costs to stakeholders

## Next growth areas

- apps and HTTP handlers
- deeper custom field and entity API coverage
- more realistic integration retries and observability
- optional React or admin-facing UI work
